## ディレクトリ構成

### world-clock-api

「3章 スラッシュコマンドを作る」で構築する世界時計のバックエンドです。

### notify-for-working

「4章 データベースを使う」で構築する出退勤コマンドのバックエンドです。

### receipt-writer

「5章 ファイルを処理する」で構築する受領証書き込みBotのバックエンドです。
